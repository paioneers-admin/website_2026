# Paioneers Brand Guidelines

## Color Palette

### Core Colors

| Role            | Hex       | Usage                              |
| --------------- | --------- | ---------------------------------- |
| Cream           | `#FAFFEF` | Page background, light surfaces    |
| Dark Teal       | `#032C2D` | Text, headings, dark backgrounds   |
| Teal            | `#0F7A99` | Primary accent, links, interactive |
| Periwinkle      | `#A9B8FF` | Gradient midpoints, highlights     |
| Orange          | `#F37C38` | CTA accent, gradient emphasis      |
| Light Blue      | `#5B8DEF` | Supporting accent                  |
| Gray            | `#8FA097` | Muted / secondary text             |

## Gradients

- **Primary CTA:** Orange → Periwinkle → Teal
- **Heading accent:** Orange → Periwinkle _or_ Teal → Periwinkle
- **Vertical divider:** Teal → Periwinkle → Orange

## Typography

| Role              | Font           | Weight        | Notes                          |
| ----------------- | -------------- | ------------- | ------------------------------ |
| Body / Headings   | IBM Plex Sans  | 200–700       | Default body font              |
| Labels / Nav / UI | IBM Plex Mono  | 400–600       | Uppercase, wide letter-spacing |

### Type Scale

| Element          | Size                     | Weight | Letter-spacing |
| ---------------- | ------------------------ | ------ | -------------- |
| Hero headline    | clamp(42px, 5vw, 72px)   | 300    | -0.02em        |
| Section headline | clamp(28px, 4vw, 58px)   | 700    | -0.03em        |
| Large display    | clamp(72px, 8vw, 120px)  | 200    | —              |
| Subhead / lead   | 18–20px                  | 300    | —              |
| Nav / labels     | 13px mono                | 500    | 0.08–0.15em    |
| Small tags       | 10–12px mono             | 500    | 0.12–0.15em    |

## UI Patterns

- **CTA buttons:** Mono, uppercase, 0.08em tracking, 1px dark teal border, 12px 24px padding. Hover fills dark teal with cream text.
- **Gradient buttons:** Animated gradient border (Orange → Periwinkle → Teal); gradient fills on hover.
- **Dark sections:** Dark Teal `#032C2D` background, cream text.
- **Frosted header:** Cream at 92% opacity + 24px backdrop blur.
